# INT-000 US-004: 貸出中の備品一覧を確認する

## As a
- 備品管理者 / 利用者

## I want
- トップページで現在貸出中の備品と備品全体の一覧を確認したい

## So that
- どの備品が誰に貸し出されているかを一目で把握でき、備品の行方不明を防止できる

## Acceptance Criteria
- [ ] トップページに備品一覧が表示される
- [ ] 各備品に現在の貸出状況（貸出中 / 返却済み）が表示される
- [ ] 貸出中の場合、借りた人の名前と貸出開始日時が表示される
- [ ] 備品名をクリックすると備品詳細ページへ遷移できる
