# INT-000 US-001: 備品を登録して QR コードを発行する

## As a
- 備品管理者

## I want
- 備品の名称・説明・数量を入力して登録し、QR コードを発行したい

## So that
- 各備品に QR コードラベルを貼り付け、スキャンによる貸出・返却管理を開始できる

## Acceptance Criteria
- [ ] 備品一覧ページから「備品を登録」ボタンで登録フォームを表示できる
- [ ] 名称（必須）・説明（任意）・数量（必須、1以上）を入力して保存できる
- [ ] 登録後、その備品の QR コードが表示される（QR コードには備品詳細ページの URL が埋め込まれている）
- [ ] 登録した備品が備品一覧に表示される
