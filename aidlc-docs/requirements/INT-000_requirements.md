# INT-000 Requirements

## 機能要件一覧

### FR-001: タスク管理（CRUD + 完了チェック）
- タスクを新規作成できる（タイトル・説明・締め切り日・優先度・プロジェクト・ステータスを入力）
- タスクを一覧表示できる（フィルタリング・ソート対応）
- タスクを編集できる（全フィールド変更可）
- タスクを削除できる
- タスクに完了チェック（完了/未完了）を付けられる
- タスクのステータスを「To Do」「In Progress」「Done」の3つから選択できる

### FR-002: プロジェクト管理
- プロジェクトを新規作成できる（名前・説明）
- プロジェクトを一覧表示できる
- プロジェクトにタスクを紐付けられる
- プロジェクト単位でタスクをフィルタリングして表示できる
- プロジェクトを削除できる（配下タスクは未割り当てになる）

### FR-003: 締め切り日・優先度の設定
- タスク作成・編集時に締め切り日（日付）を設定できる
- タスク作成・編集時に優先度（高/中/低）を設定できる
- 優先度・締め切り日によるソートが可能

### FR-004: カンバンボード表示
- タスクをステータス（To Do / In Progress / Done）別にカラム表示できる
- カラム内でタスクカードをドラッグ&ドロップでステータス変更できる（PoCではボタン操作でも可）
- 各カードにタイトル・優先度・締め切り日を表示する

### FR-005: 備品QR貸し出し/返却管理
- 備品を登録できる（名前・型番・説明）
- 備品ごとにQRコードを生成・表示できる（備品IDを埋め込む）
- QRコードをスキャン（またはURLアクセス）すると、貸し出し/返却の操作画面が表示される
- 貸し出し記録を残せる（備品ID・借用者名・貸し出し日時）
- 返却操作で返却日時を記録できる
- 備品ごとの貸し出し履歴を閲覧できる
- 現在の貸し出し状況（貸出中/返却済）を一覧で確認できる

## 入出力・制約

| 項目 | 内容 |
|------|------|
| 入力 | フォーム入力（テキスト・日付・セレクト）、QRスキャン（カメラ or URL） |
| 出力 | タスク一覧・カンバンボード・備品一覧・QRコード画像・貸し出し履歴 |
| DB制約 | タスクのタイトルは必須、備品名は必須、貸し出しは1備品につき同時1件のみ |
| 認証 | なし（PoCのため） |

## 依存関係
- Drizzle ORM + Neon PostgreSQL（既存インフラを共有）
- Next.js App Router（既存フレームワーク）
- shadcn/ui + Tailwind CSS（既存UIライブラリ）
- QRコード生成ライブラリ（`qrcode` または `react-qr-code`）
- QRコードスキャンライブラリ（`html5-qrcode` または `react-qr-scanner`、PoC範囲ではURL直接アクセスでも代替可）
