# INT-000 Intent

## Purpose
小規模チーム向けの「タスク・プロジェクト管理 + 備品QR管理」ツールを構築する。
既存のコメント投稿アプリ（Brownfield）をベースに、タスク管理画面と備品管理画面を追加する。
PoCとしてスピーディに動作するアプリケーションを提供することを最優先とする。

## Success Criteria
- タスクのCRUD（作成・編集・削除・完了チェック）が動作すること
- プロジェクト単位でタスクを整理・閲覧できること
- タスクに締め切り日・優先度を設定できること
- カンバンボード（To Do / In Progress / Done）でタスクを可視化できること
- QRコードを使って備品の貸し出し・返却を管理できること
- 既存のコメント画面（`/`）は引き続き動作すること

## Scope

### 対象（In Scope）
- タスクCRUD画面（`/tasks`）
- プロジェクト管理（プロジェクト作成・タスクへの紐付け）
- タスクの締め切り日・優先度設定
- カンバンボード表示（`/tasks/board`）
- 備品管理画面（`/equipment`）
  - 備品の登録・一覧表示
  - QRコード生成（備品ごと）
  - QRコードスキャンによる貸し出し・返却記録
- DBスキーマ追加（Drizzle ORM + Neon PostgreSQL）
- Next.js App Router + shadcn/ui + Tailwind CSS によるUI実装

### 対象外（Out of Scope）
- ユーザー認証・認可（ログイン機能なし）
- 通知・メール送信
- 外部カレンダー連携
- モバイルアプリ（Webのみ）
- 非機能要件（パフォーマンス・セキュリティ・可用性の詳細設計）

## Brownfield Context
- 既存コメント投稿アプリ（`/` ページ）は変更せず維持する
- DBはNeon PostgreSQL（Drizzle ORM）を共有使用する
- 既存テーブル（`comments`等）には影響を与えない
